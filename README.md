**docker-compose MUST be installed**

[> CLIENT](localhost:4200)

[> SERVER](localhost:4200)
[> ADMIN](localhost:4200/admin)

```bash
# yarn = npm run

yarn start
yarn log [client|server|db]
yarn logs
yarn stop

yarn client:add bootstrap@4.1.1
yarn client:restart
yarn client:start
yarn client:stop
yarn client:shell

yarn server:add graphql
yarn server:restart
yarn server:start
yarn server:stop
yarn server:shell
```
